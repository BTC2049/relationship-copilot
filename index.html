    Setup Guide

    用戶怎麼開始用

    TG Telegram：最快 MVP，用匯出檔或 Bot 起步

    一般用戶

    1. 打開 Telegram Desktop。
    2. 進入要分析的聊天、群組或頻道。
    3. 選擇 Export chat history。
    4. 格式選 JSON，若不方便就選 HTML。
    5. 把匯出的檔案拖進本工具。

    團隊或社群主

    1. 建立 Telegram Bot。
    2. 把 Bot 加進群組並給讀訊息權限。
    3. 把 Bot Token 填到連接器設定。
    4. 之後可改成每日自動同步群內互動。

    注意：個人私訊不建議用 Bot 偷讀，應用官方匯出檔或明確授權流程。

    LN LINE：先用聊天備份，官方帳號再接 webhook

    一般用戶

    1. 在 LINE 打開指定聊天。
    2. 進入聊天設定。
    3. 選擇匯出聊天紀錄。
    4. 取得 txt 檔後拖進本工具。

    官方帳號

    1. 建立 LINE Developers Channel。
    2. 開啟 Messaging API。
    3. 設定 Webhook URL。
    4. 把 Channel Secret / Access Token 填進連接器。

    LINE Messaging API 主要服務官方帳號，不等於可以直接讀個人 LINE 私訊。

    IG IG / FB：適合商業帳號與粉專訊息

    快速使用

    1. 先從 Meta 下載你的資料，或複製對話文字。
    2. 把訊息文字貼到聊天紀錄輸入區。
    3. 選擇 IG / FB / LinkedIn 格式後解析。

    正式串接

    1. 建立 Meta App。
    2. 連接 Facebook Page 或 Instagram Business Account。
    3. 申請 Messenger / Instagram Messaging 權限。
    4. 通過審核後設定 webhook 同步訊息。

    Meta 權限審核較嚴，MVP 應先從商業帳號、粉專訊息與匯出檔做起。

    IN LinkedIn：先接 CRM、會議紀錄與匯出資料

    快速使用

    1. 匯出 LinkedIn connections 或 CRM 名單。
    2. 補上最近會議紀錄、Email 摘要或對話文字。
    3. 匯入本工具做關係健康度分析。

    正式串接

    1. 建立 LinkedIn Developer App。
    2. 設定 OAuth Redirect URL。
    3. 申請可用的 Profile / Organization 權限。
    4. 把資料回寫到 CRM 或本工具的人脈資料庫。

    LinkedIn 對私訊與關係資料限制很高，建議不要把私訊直連當第一版核心。
